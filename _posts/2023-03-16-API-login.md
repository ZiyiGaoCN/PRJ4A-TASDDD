# URL `/login`
## POST

请求验证用户的用户名和密码。如果成功，会返回`token`

### 请求体
```json
{
  "userName":"Alice",
  "password":"Encrypted String"
}
```

### 成功响应体

状态码设置成`200`

```json
{
  "info": "Succeed",
  "code": 0,
  "data": {
    "token": "[Some token]"
  }
}
```
### 失败响应体
状态码为`500`,code码为`-1`
```json
{
    "code": -1,
    "info": "[Some message]"
}
```

`info`包括但不限于: `Invalid username or password`.
