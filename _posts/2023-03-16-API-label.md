# URL `/label`
该API仅接受POST方法请求，方法请求均应当设置状态码为 405 Method Not Allowed，错误响应格式为：
```json
{
  "code": -3,
  "info": "Bad method"
}
```
## POST

请求标注任务并创建标注"答卷"。

### 请求体
```json
{
  "labelId": 1,
  "taskId": 2
}
```
`labelerId`:所属标注方的ID

`task`:所属标注任务的ID

### 成功响应体

状态码设置成`200`

```json
{
  "info": "Succeed",
  "code": 0
}
```
### 失败响应体
状态码为`500`,code码为`-1`
```json
{
    "code": -1,
    "info": "[Some message]"
}
```

`info`包括但不限于: `Already labeled this task`.
